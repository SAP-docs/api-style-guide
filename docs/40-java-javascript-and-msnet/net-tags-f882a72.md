# .NET Tags

A reference of the .NET tags that you can use in documentation comments.



We recommend the following order of tags inside a comment block:

```
/// <include />
///
/// <summary> ... </summary>
/// <param> ... </param>
/// <returns> ... </returns>
/// <exception> ... </exception>
/// <remarks> ... </remarks>
/// <see />
/// <example> ... </example>
```
